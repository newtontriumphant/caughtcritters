# Caught Critters

## About

This website was made to share images and videos of wildlife "caught" on film in Marin County, CA.

## Our sighted animals
### (as of Mar. 12, 2021)

* Deer
* Coyotes
* Bobcats
* Raccoons
* Squirrels
* Skunks
